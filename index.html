INTRODUCCIÓN A GIT

INSTALAR GIT

En sistemas Debian, Ubuntu, Mint:

#apt get install git

MANOS A LA OBRA !

CONFIGURACIÓN

Parámetros básicos 

git config --global user.name "Nombre de usuario"
git config --global user.mail "correo@whatever.com"
 git config --list

IT'S ALL ABOUT REPOS

PIMEROS PASOS

Iniciar un repositorio 

 git init

Clonar un repositorio 

 git clone url_repositorio

Git conoce reconoce el proyecto en el que estás trabajando en función del lugar donde te encuentres al ejecutar los comandos

FUNCIONAMIENTO DE GIT

git add
git commit
Directorio de trabajo → INDEX → HEAD

MANTENER LOS ARCHIVOS DE NUESTRO REPOSITORIO

git add nombre_archivo
git add mi archivo.*
Eliminar un archivo del repositorio 
git add -u

MANTENER LOS ARCHIVOS DE NUESTRO REPOSITORIO

git status
INDEX → HEAD 
git commit nombre_archivo
git commit -m "Comentario de los archivos"
git commit -am "Comentario de los archivos"

SINCRONIZANDO REPOSITORIOS 

git remote
git remote -v 
origin git@github.com:cecimerelo/GIT.git (fetch) 
origin git@github.com:cecimerelo/GIT.git (push)
